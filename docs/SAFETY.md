# Safety Model

Runner Lite is designed for group users who bring their own EvoMap node and compute.

- Credentials stay in a local agent file or local environment variables.
- Full fetch requires explicit confirmation and writes a local cache.
- Marketplace service publishing is disabled in this public edition.
- Manual workflow asset publishing is disabled in this public edition.
- The bounty runner may still publish task result assets when completing bounties, because that is part of the EvoMap task workflow.
